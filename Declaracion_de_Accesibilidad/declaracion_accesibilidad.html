Declaracion de accesibilidad

Accesibilidad en MeDiccionary

En MeDiccionary, nos dedicamos a asegurar que nuestro sitio web sea accesible para todas las personas, sin importar sus capacidades. Trabajamos activamente para cumplir con las Pautas de Accesibilidad para el Contenido Web (WCAG) 2.1, nivel AA, así como con lo establecido en el Real Decreto 1112/2018.

Medidas de Accesibilidad Implementadas

  • Identificación del idioma: Todas nuestras páginas están etiquetadas correctamente con el atributo de idioma, facilitando el uso de lectores de pantalla.
  • Texto alternativo: Proporcionamos descripciones alternativas en las imágenes para hacer accesible el contenido visual a personas con discapacidades visuales.
  • Navegación clara: Contamos con un mapa del sitio que simplifica la navegación por todo el contenido disponible en HealthHub.
  • Estructura semántica: Utilizamos etiquetas semánticas de HTML5 para que la estructura del contenido sea fácilmente comprensible para las tecnologías de asistencia.
  • Contraste optimizado: El diseño del sitio está adaptado para asegurar un contraste adecuado que mejore la legibilidad, beneficiando a personas con baja visión.
  • Accesibilidad por teclado: Todos los elementos interactivos son accesibles mediante el teclado, favoreciendo a usuarios con discapacidades motoras.

Conformidad con los Estándares

Nuestro sitio web cumple mayormente con las pautas WCAG 2.1, nivel AA, y estamos trabajando en mejorar algunos aspectos para garantizar una accesibilidad plena.

Si tienes dificultades para acceder a cualquier parte de nuestro sitio o quieres hacernos llegar tus comentarios sobre la accesibilidad, puedes contactarnos a través del Formulario de ayuda al cliente.
